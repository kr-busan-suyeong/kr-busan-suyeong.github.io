# 수영구 · 부산광역시

> 수영구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-busan-suyeong.github.io/**

수영구는 광안대교와 광안리해수욕장으로 대표되는 부산의 대표 해양·야경 관광지입니다. 구 이름은 조선시대 이곳에 자리했던 경상좌도 수군절도사영(좌수영)에서 따온 것으로, '수군'의 물 수(水)와 '절도사영'의 진영 영(營)을 합쳐 '수영'이라 불렀습니다. 1.4km에 이르는 광안리 백사장과 밤마다 색을 바꾸는 광안대교 조명, 그리고 국내 최초의 수변공원인 민락수변공원이 어우러져 사시사철 관광객과 시민의 발길이 끊이지 않습니다. 매년 가을 이곳 바다를 배경으로 부산불꽃축제가 열리고, 봄에는 좌수영어방놀이에서 비롯된 광안리 어방축제가 펼쳐집니다. 와이어 공장을 개조한 복합문화공간 F1963과 망미동 비콘그라운드처럼 옛 산업시설을 문화공간으로 되살린 명소들도 수영구만의 개성을 더합니다.
